# carmenpelayo.github.io

Personal academic/professional website, generated by a single Python script.

## Edit & rebuild
All content (bio, publications, experience, projects, skills) lives in plain
Python dictionaries at the top of `build.py`. After editing:

    python3 build.py

This regenerates `index.html` and `assets/style.css` / `assets/script.js`.
The hero chart is computed in Python from the headline finding of
"Measuring Technological Progress in Real Time with ArXiv" (Pelayo, 2025).

## Deploy on GitHub Pages
1. Create a public repo named exactly `carmenpelayo.github.io`.
2. Push the contents of this folder (index.html, assets/, build.py, README.md).
3. In the repo: Settings -> Pages -> Source: "Deploy from a branch",
   Branch: `main`, folder `/ (root)`. Save.
4. The site goes live at https://carmenpelayo.github.io within ~1 minute.

## Before going live
- Add your photo as `assets/photo.jpg` (square crop works best; it renders
  as a circle). If absent, the site simply hides the photo.
- `assets/Carmen_Pelayo_CV.pdf` is linked from the hero "CV (PDF)" button —
  replace it whenever you update your CV.
- Update the two publication links in `build.py` with the exact report URL
  once you have it.
